__for first launch__
```console
foo@bar:~$ cd frontend
foo@bar:~$ npm install
```

open two terminals and run

1.  __frontend launch__
    ```console
    foo@bar:~$ cd frontend
    foo@bar:~$ npm run dev
    ```

2.  __backend launch__
    ```console
    foo@bar:~$ python manage.py runserver
    ```
